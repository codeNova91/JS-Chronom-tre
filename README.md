# ⏱️ Chronomètre Dynamique

Une application utilitaire précise développée en **JavaScript** pur pour mesurer le temps avec des fonctions de démarrage, d'arrêt et de réinitialisation.

## Fonctionnalités
- **Démarrer** : Lancer le compteur.
- **Arrêt** : Mettre le temps en pause.
- **Réinitialiser** : Remettre le compteur à zéro.
- Mise à jour fluide du DOM chaque seconde.

##  Technologies utilisées
- **HTML5**
- **CSS3** (Flexbox pour le centrage)
- **JavaScript** (setInterval & Timing events)
